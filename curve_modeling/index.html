Curve modeling helpers#

Simple modeling helpers for 2d curves, that are trying to preserve sewings order on edited sewing pattern.

Subdivide Segment GM#

Subdivide 2d curve segment, while preserving sewing order on edited pattern. Located under curve edit menu: 'W'- key. In video above you can see sewing order is broken, if you delete vert the traditional way.

Delete vert GM#

Remove 2d curve verts, while preserving sewing order on edited pattern. Located under 'X' - key. On video above you can see sewing order is broken, if you delete vertex the traditional way.
