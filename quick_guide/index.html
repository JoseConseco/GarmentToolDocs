Garment Tool supports only cyclic (closed) 2d Bezier curves. To create one use Shift+A > Curve -> Bezier Circe, and make any shape you wish from it.
For T-shirt, add four bezier circles, and model them into sewing patterns, like on image below:

Place Sewing Patterns next to your character.

To create garment click +(Plus) icon, in the Garment Tool Panel.
